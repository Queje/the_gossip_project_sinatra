## This project in ruby use the sinatra gem to build and MVC website.
### the objective is to build ruby files using one class per file and every methods must be in a class. The sinatra gem is used to create a local server.
### M: methods are written in the gossip.rb file 
### V: We used erb files placed in a views folder to prepare the html pages showed to the users.
### C: The controller links the different pages in erb and in programs in rb.
### This program can be launch using the shotgun gem in with the port 4567, shotgun -p4567
